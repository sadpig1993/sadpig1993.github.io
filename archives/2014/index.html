nginx代理proxy_pass设置

今天在服务器上装了node和express

用forever守护进程自启动

express用的8080端口，然后用nginx反向代理把域名代理到了127.0.0.1:8080了。

server
{
listen 80;
server_name node.miyijia.com;
location / {
proxy_pass http://127.0.0.1:8080;
proxy_redirect off;
proxy_set_header X-Real-IP $remote_addr;
proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
}
}
